Паттерны или шаблоны разработки - это общие способы решения задач и проблем
Декораторы в питоне - также один из паттернов проектирования

Singleton (Одиночка) - это шаблон предоставления глобального доступа к состоянию,
гарантируется, что объект всегда будет один и тот же.
Singleton - это класс, который гарантирует, что во всей системе присутствует только
1 объект этого класса, и кто бы и откуда бы не вызывал этот объект он всегда будет
один и тот же 

Нужен для одной точки доступа к ресурсам/данным и для того
чтобы ресурсоемкие задачи сделать 1 раз

Monostate - это шаблон предоставления глобального доступа к состоянию,
объекты при этом могут быть разными.

+ плюсы: 1 раз выполняем тяжелые задачи, имеем 1 вход для всей системы
- минусы: общесистемная глобальная переменная. Таким образом если сломать эту
глобальную переменную при работе с ней в каком-то из модулей,
то испортится к ней доступ ВО ВСЕЙ СИСТЕМЕ, что небезопасно. Потому некоторые
люди дажи называют синглтон АНТИПАТТЕРНОМ и не рекомендуют его использовать

Модуль в python - это синглтон!
То есть при импорте модуля объект с именем модуля создается один раз, и мы из части
системы получаем глобальную точку входа...ИМЕННО ПОЭТОМУ ВЫГОДНО ИСПОЛЬЗОВАТЬ МОДУЛИ
Таким образом 80% случаев где требуется синглтон в питоне покрываются модулями!!!
(Для сравнения в Java нет модулей, и там придется писать класс)
Потому нам никуда от них не деться, даже если считать синглтон анти-паттерном

В каких случаях все-таки нужно писать отдельные классы-паттерны 
синглон или моностейт?
* Когда мы хотим инкапсулировать данные
* Когда хотим для синглона перегрузить магические методы (например если 
инкапсулируется работа с пулом потоков или с пулом соединений с бд, и мы хотим
реализовать метод __iter__, который будет перебирать например свободные
веб-драйвера для гугл-хрома)
* Когда хотим реализовать наследование

Дальнейшее изучение паттернов программирования необходимо проводить
отталкиваясь от реальных проблем, которые решают различные паттерны.
Иначе, это будет пустой тратой времени.
